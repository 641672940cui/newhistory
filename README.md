基于 HTML/CSS/JS 的通用题库引擎，**题库数据与程序完全分离**。

## 🌟 特性

- 📖 **选择题分区练习**：160题分8个专区，进度跟踪
- 📜 **材料题四步解题法**：读材料→联所学→析因果→得结论
- 📝 **智能错题本**：自动收集错题，答对自动移除
- 🖨️ **错题打印**：一键打印错题本，带正确答案高亮
- 🔥 **连击鼓励**：连续答对有特殊鼓励弹窗
- 📝 **支持填空题**：JSON中配置 `type: "fill_blank"` 即可
- 🧩 **题库JSON独立**：AI可安全生成/更新题库，不影响程序
- 📱 **移动端优先**：响应式设计，触屏优化
- 🔗 **索引页统一管理**：多科目一页入口
